## Technical Stack
- **Languages:** Python (Pandas, Scikit-learn), R (Shiny), SQL (Postgres)
- **Automation:** n8n (Workflow Orchestration), REST APIs, Docker
- **AI/ML:** LLM-powered triage, Crude & Credibility-blended modeling

## Key Projects

### 1. AI-Powered Automated Claims Adjuster
An event-driven microservice designed to eliminate latency in the FNOL process.
- **Logic:** Uses n8n to ingest payloads and OpenAI (GPT-4o-mini) to output severity scores and reserve estimates.
- **Storage:** Structured data is committed to a PostgreSQL warehouse.
- **Visualization:** Real-time analytics rendered via R Shiny.

### 2. Actuarial Experience Study - Lapse MVP
A reproducible framework for calculating insurance policy lapse rates.
- **Methods:** Calculates crude lapse rates by duration and applies credibility-blending ($Z = \frac{n}{n + k}$).
- **Output:** Generates 95% confidence interval plots and structured data dictionaries for audit readiness.
